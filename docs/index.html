ご利用にあたって（重要）

本サイトは、航空法・小型無人機等飛行禁止法とは別に、各自治体・施設管理者が公園・海岸・施設等に定める独自ルールを地図上に整理した参考情報です。飛行の可否を断定するものではありません。

  • 情報は自動収集・手動確認が混在し、正確性・最新性を保証しません。
  • 飛行可否の最終判断は、必ず各管理者および国土交通省のDIPSで確認してください。
  • グレーは「自治体独自の規制は確認されず（国の航空法等のみ）」の意味です。飛行可否は国の規制とDIPSで確認してください。
  • 自治体規制の分類は各自治体の公式ページを一次情報として確認しています。国の空域・RZ/YZ等は地理院地図と同一データの参考表示です。

# 自治体ドローン規制マップ（drone-jichitai-map）

ドローンの飛行可否は、航空法・小型無人機等飛行禁止法・空域（＝国の規制）だけでは決まりません。実際には各自治体や施設管理者が、公園・海岸・海水浴場・港湾・漁港・河川・ダム・観光地・スポーツ施設などに独自ルールを敷いており、一元化されていません。それを国の空域情報と重ねて地図に載せるプロジェクトです。

**サイト: https://nari4d.github.io/drone-jichitai-map/**

## ⚠ 免責事項

- 本サイトは**参考情報**であり、飛行の可否を断定するものではありません。
- 情報は自動収集を含み、**正確性・最新性を保証しません**。
- 飛行可否の**最終判断は必ず各管理者および国土交通省の DIPS で確認**してください。
- グレー表示は「規制なし」ではなく「**自治体独自の規制は確認されず（国の規制のみ）**」の意味です。

## データ方針

規制の分類・出典は、**各自治体・省庁の公式ページ（一次情報）のみ**を根拠とします。ドローン系ブログ・行政書士サイト・ニュース記事等の二次情報は使いません（誤りが確認されたため）。公式に明文が確認できない場合は、推測で埋めず「確認されず」として扱います。

## データの出典とクレジット

| データ | 出典 |
|---|---|
| 市区町村境界 | **国土交通省 国土数値情報（行政区域 N03, 2025年）** https://nlftp.mlit.go.jp/ksj/ ※クレジット記載が必要 |
| 背景地図タイル | 国土地理院（地理院タイル） |
| 空港等周辺空域・管制圏等 | 国土交通省航空局（地理院地図経由） |
| RZ/YZ（小型無人機等飛行禁止法） | 警察庁（地理院地図経由） |
| 人口集中地区（DID 令和2年） | 総務省統計局 |
| 各自治体の規制 | 各自治体の公式ページ（`data/regulations_*.json` の `sources` に個別記載） |
| 地名検索 | OpenStreetMap / Nominatim（地図移動用途のみ。規制の出典ではありません） |
